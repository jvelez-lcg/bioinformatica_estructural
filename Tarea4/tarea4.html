1. Mutaciones puntuales en proteínas

El interés por predecir la estructura terciara de proteínas con mutaciones puntuales ha crecido desde que el mundo entró a la era de los “Omicas”. Ahora es posible conocer millones de fenotipos moleculares (tales como los SNPs) y es válido preguntarse cómo luciría la estructura secundaria de estos; ¿cuál es el impacto de una mutación en alguna posición de la estructura protéica? Si existe un cambio en la estructura, ¿de qué forma afecta a la función?

Para observar de manera gráfica los efectos en la estructura que pueden tener las mutaciones puntuales en las proteínas variamos la secuencia de P1 en 6 sitios distintos, es decir, generamos seis “alelos” o “mutaciones”.

5. Contactos entre residuos:

Como última prueba se escribió un programa que calculara para cada mutante y la wildtype el número de contactos con los residuos vecinos en un radio de \(15 angstroms\) (sin contar las interacciones de los hidrógenos). Para poner cada uno de estos contactos en contexto se calculó la distribución de los contactos de todos los residuos. El siguiente código fue ejecutado manualmente para cada mutante.

5.1 Obtencion de matrices de contacto y distancias.

Mostrar código

import numpy as np
import pandas as pd
import seaborn as sns
from Bio.PDB import *
import matplotlib.pyplot as plt
def calc_dist_matrix(chain_one, chain_two) :
    """Returns a matrix of C-alpha distances between two chains"""
    
    def calc_residue_dist(residue_one, residue_two) :
        """Returns the C-alpha distance between two residues"""
        diff_vector  = residue_one["CA"].coord - residue_two["CA"].coord
        return np.sqrt(np.sum(diff_vector * diff_vector))
    
    answer = np.zeros((len(chain_one), len(chain_two)), np.float)
    for row, residue_one in enumerate(chain_one) :
        for col, residue_two in enumerate(chain_two) :
            answer[row, col] = calc_residue_dist(residue_one, residue_two)
    return answer
def get_contact_matrix_of_residues1(chain_one, chain_two,distance) :
    """Returns a matrix of Contacts between two chains"""
    
    def at_least_one_atom_nearby(residue_one, residue_two,distance) :
        """Returns True if at least one pair of atoms between the residues is at a given distance.
        Otherwise, return False."""
        for atom1 in residue_one.get_atoms():
            for atom2 in residue_two.get_atoms():
                atom_distance = np.sqrt(np.sum(np.power(atom1.coord - atom2.coord, 2)))
                if atom_distance < distance: return True
        return False
    
    
    answer = np.zeros((len(chain_one), len(chain_two)), np.int)
    for row, residue_one in enumerate(chain_one) :
        for col, residue_two in enumerate(chain_two) :
            answer[row, col] = at_least_one_atom_nearby(residue_one, residue_two,distance)
    np.fill_diagonal(answer,0)
    return answer
def get_contact_matrix_of_residues2(structure, distance):
    atoms = Selection.unfold_entities(structure, 'A')
    residues_neighbors = NeighborSearch(atoms).search_all(distance, 'R')
    
    num_residues = len(Selection.unfold_entities(structure, 'R'))
    contact_matrix = np.zeros((num_residues,num_residues), dtype = np.int)
    for pair in residues_neighbors:
        i, j = pair[0].get_id()[1]-1, pair[1].get_id()[1]-1
        contact_matrix[i][j], contact_matrix[j][i] = 1, 1
    return contact_matrix
